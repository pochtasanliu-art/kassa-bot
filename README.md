# 💰 Бот-Касса

Telegram бот для учёта наличных операций. Каждый чат ведёт свою отдельную кассу.

## Как пользоваться

**Добавить приход:**
```
+50000 приход наличные
+120000 Алексей оплата
```

**Добавить расход:**
```
-30000 выдача Inga
-15000 расходы на офис
```

**Команды:**
- `/касса` — текущий остаток
- `/история` — последние 10 операций
- `/обнулить` — сбросить кассу

---

## Деплой на Railway.app (бесплатно)

1. Зарегистрируйся на https://railway.app
2. Нажми **New Project → Deploy from GitHub**
3. Загрузи файлы бота (bot.py, requirements.txt, Procfile)
4. В разделе **Variables** добавь переменную:
   - `BOT_TOKEN` = твой токен от BotFather
5. Нажми **Deploy** — бот запустится автоматически

---

## Деплой на Render.com (альтернатива)

1. Зарегистрируйся на https://render.com
2. Новый сервис → **Background Worker**
3. Загрузи файлы
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python bot.py`
6. Добавь переменную `BOT_TOKEN`
7. Deploy!
